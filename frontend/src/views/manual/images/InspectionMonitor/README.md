# 検査モニタ — スクリーンショット用フォルダ

操作説明書 `frontend/src/views/manual/docs/inspection-monitor_ja.md` から参照する画像を、このフォルダに配置してください。

## 推奨ファイル名一覧

| ファイル名 | 内容 |
|-----------|------|
| `01_overview.png` | 画面全体（KPI・稼働カード・不良・能率・確定実績） |
| `02_menu_navigation.png` | MES → モニタリング → 検査モニタ |
| `03_header_toolbar.png` | 生産日・自動更新・全画面・更新ボタン |
| `04_kpi_row.png` | KPI サマリー行（稼働中・能率・不良率等） |
| `05_runtime_cards.png` | 検査工程稼働カード（経過・検査員・ステータス） |
| `06_defect_table.png` | 不良（項目別）テーブル |
| `07_efficiency_table.png` | 検査員能率ランキング |
| `08_history_cards.png` | 本日の確定実績カード |
| `09_next_assign_workflow.png` | 次製品指定フロー図（§10） |
| `09_next_assign_workflow.mmd` | 上記 Mermaid ソース |
| `10_next_assign_drawer.png` | 次製品指定ドロワー（稼働中/未稼働区分） |
| `11_next_assign_dialog.png` | 次製品指定ダイアログ |

## 撮影のヒント

- 解像度：1280px 以上（全画面表示のスクショも推奨）
- 稼働データ：複数検査員が稼働中の状態で KPI・カードを撮影
- **通信断**：該当カードがある場合は `05` に含める
- **次製品指定**：ドロワー・ダイアログは別途撮影
