# 検査工程 — 生産性分析 画面 操作説明

| 項目 | 内容 |
|------|------|
| メニュー | **MES** → **実績分析項目** → **生産性分析** → **検査生産性** |
| 画面名（画面上） | **検査工程 — 生産性分析** |
| 操作説明画面 | 画面上部 **マニュアル** ボタン → 左メニュー **MES関連** → **検査工程 — 生産性分析** |

本書は、検査工程の **生産数・能率・不良率・稼働時間** を期間単位で集計・可視化し、検査員・製品別の比較やレポート出力を行う手順を説明します。  
実績の入力は **[検査実績収集]** または **[検査実績収集登録]** で行います。

## 1. 画面の目的とできること

検査工程 — 生産性分析では、指定した **集計期間** の検査 MES 実績を横断的に分析します。

| 機能 | 説明 |
|------|------|
| 期間集計 | 開始日〜終了日で `データベース` を集計 |
| フィルタ | 検査員・製品・未確定セッション含有の絞り込み |
| KPI | 確定セッション数、生産数、不良、総合能率、正味稼働 |
| チャート | 日別推移、検査員別・製品別の棒グラフ |
| ランキング | 溶接工程あり/なし別の検査員能率、製品別検査員順位 |
| 指標表 | シフト・休憩・ロス・作業率・稼働率・不良内訳（KT09） |
| 明細 | セッション単位の一覧と **CSV エクスポート** |
| レポート | 全体またはセクション別の **印刷**（ブラウザ印刷） |

**デフォルト期間**

- 初回表示時：**直近 30 日**（本日を含む 29 日前 〜 本日、JST）

## 2. 画面へのアクセスと操作説明の開き方

### 2.1 画面を開く

1. Smart-EMAPs にログインします。
2. 左サイドメニュー **MES** → **実績分析項目** → **生産性分析** を展開します。
3. **検査生産性** をクリックします。

![メニューから検査生産性を開く](./images/InspectionProductivity/02_menu_navigation.png)

### 2.2 本操作説明（マニュアル）を開く

1. 画面上部 **マニュアル** ボタン → 左メニュー **MES関連** → **検査工程 — 生産性分析**

## 4. フィルタ（集計条件）

![フィルタバー](./images/InspectionProductivity/03_filters.png)

| 項目 | 説明 |
|------|------|
| **期間** | 日付範囲（開始 ～ 終了）。変更すると約 350ms 後に自動再集計 |
| **検査員** | `（すべて）` または特定検査員（期間内に実績がある候補） |
| **製品名** | `（すべて）` または検査対象製品（CD 末尾 `1` 等のフィルタ済み） |
| **未確定を含む** | OFF（既定）：`production_completed_check = 1` のみ集計<br>ON：生産中・未確定セッションも含む |

**ヘッダー右の期間表示**

- データ取得後、`YYYY-MM-DD ～ YYYY-MM-DD` が表示されます。

**更新ボタン**

- フィルタ変更に加え、手動で **更新** を押すと即時再取得できます。

## 5. KPI サマリー

![KPI カード](./images/InspectionProductivity/04_kpi.png)

| カード | 意味 | 補足（hint） |
|--------|------|--------------|
| **確定セッション** | 確定済み件数 | `全 n 件`（未確定含む総セッション） |

| **不良数** | 不良合計 | 不良率（%） |
| **総合能率** | 期間全体の能率 | 本 / 時間 |
| **正味稼働** | 正味稼働時間（分換算表示） | 停止時間の合計 |

## 9. 検査員平均能率ランキング（溶接あり / なし）

![溶接ランキング](./images/InspectionProductivity/08_weld_rank.png)

製品工程 BOM（`product_process_bom`）の **溶接工程フラグ** に基づき、検査実績を 2 系統に分けて検査員の平均能率を順位付けします。

| パネル | 対象 |
|--------|------|
| **溶接工程なし製品** | BOM に溶接がない製品の検査のみ |
| **溶接工程あり製品** | BOM に溶接がある製品の検査のみ |

**検査員名をクリック** → [§14](#14-検査員別製品一覧ダイアログ) の製品一覧ダイアログが開きます。

**注意**

- BOM に溶接登録がない場合、右パネルに警告が表示されます。
- 該当実績がない場合は「対象データがありません」と表示されます。

## 11. 検査員別指標（時間・不良内訳）

データがある場合のみ、次の 2 テーブルが表示されます。

### 11.1 時間 / 生産

![検査員別指標・時間](./images/InspectionProductivity/10_metrics_time.png)

| 列 | 意味 |
|----|------|
| シフト | シフト時間（時間） |
| 休憩 | 休憩時間 |
| ロス時間 | 停止（段替・待ち等） |
| 作業すべき時間 | シフト − 休憩 − ロス |
| 作業時間 | 正味稼働相当 |
| 作業率 | 作業時間 ÷ 作業すべき時間 |
| 検査総数 | 生産本数 |
| 能率 | 本/時 |
| 稼働率 | 稼働率（%） |

末尾に **応援**（該当時）・**合計** 行があります。

### 11.2 不良内訳（KT09）

![検査員別指標・不良](./images/InspectionProductivity/11_metrics_defect.png)

- 工程別不良項目（KT09）の列ごとに検査員別数量を表示
- 右端に **不良合計** 列

## 13. レポート出力（印刷）

![レポートメニュー](./images/InspectionProductivity/13_report_menu.png)

ヘッダーの **レポート** ドロップダウンから、ブラウザの印刷ダイアログで出力します。分析データ読込後のみ有効です。

| メニュー項目 | 内容 |
|--------------|------|
| **印刷（全体）** | 全セクション一括 |
| **日別推移（印刷）** | 生産数・能率の推移 |
| **日別推移（検査員別・一括印刷）** | 検査員ごとに日別ページを分割 |
| **検査員別（印刷）** | 検査員別サマリー |
| **検査員別指標表（印刷）** | 不良内訳・時間・能率の表 |
| **検査員別製品別（検査員別・一括印刷）** | 検査員ごとの製品一覧 |
| **製品別（印刷）** | 製品別サマリー |
| **検査員平均能率ランキング（印刷）** | 溶接あり / なし比較 |
| **製品別 · 検査員能率ランキング（印刷）** | 製品単位の順位表 |

印刷時はチャートを画像として埋め込んでから出力します。用紙向き・余白はブラウザの印刷設定で調整してください。

## 15. 権限とデータ範囲

| 項目 | 内容 |
|------|------|
| メニュー権限 | `MES_ACTUAL_ANALYSIS_PRODUCTIVITY_INSPECTION` |
| API | ログイン必須（`verify_token_and_get_user`） |
| 集計対象 | `inspection_management` の `production_day` が期間内 |
| 既定 | **確定済み** のみ（未確定はチェック ON で含有） |
| 上限 | 1 回の取得で最大 **5000** セッション |

Excel/CSV 取込データは本画面の集計 API とは別管理です。MES 実績（検査収集・登録）が主な入力源です。

## 16. 指標の計算方法

### 16.1 能率（本/時）

```
能率 = 生産数 ÷ 正味稼働時間[時間]
```

- 正味稼働時間は `mes_net_production_sec` 等から算出
- 表示は原則 **整数**（指標表の一部は小数 1 位）

### 16.2 不良率

```
不良率(%) = (不良数 ÷ 生産数) × 100
```

小数第 1 位。生産数 0 のとき `—`

### 16.3 総合能率（KPI）

期間内の生産数合計 ÷ 正味稼働時間合計

### 16.4 検査員別パネル平均能率

全検査員の生産合計 ÷ 正味稼働合計（加重平均）

### 16.5 溶接あり / なしの判定

`product_process_bom` で製品に溶接工程が登録されているかで分類します。

## 17. よくあるメッセージと対処

| メッセージ（要約） | 対処 |
|--------------------|------|
| 集計期間を選択してください | 期間ピッカーで開始・終了を指定 |
| 集計期間を指定してください | 同上 |
| 分析データの取得に失敗しました | ネットワーク・期間の妥当性を確認し **更新** |
| 出力する分析データがありません | 先に集計が成功しているか確認 |
| 印刷できる検査員別指標データがありません | 期間・フィルタを見直す |
| レポート出力に失敗しました | 再試行。チャート描画完了後に印刷 |
| 製品工程BOMに溶接工程の登録がありません | マスタで溶接フラグを確認 |
| 対象データがありません | 期間内に該当カテゴリの実績なし |

## 18. 運用上の注意

1. **確定実績を基準に分析**  
   既定では確定済みのみ集計します。進行中を含めたい場合のみ **未確定を含む** を ON にしてください。

2. **期間の長さ**  
   長期間（数千セッション）では取得に時間がかかることがあります。必要に応じて検査員・製品で絞り込んでください。

3. **能率比較の前提**  
   正味稼働時間が 0 のセッションは能率ランキングから除外されます。データ品質（開始/終了・停止入力）を確認してください。

4. **溶接分類**  
   BOM 未登録・フラグ誤りはランキングの分類に影響します。マスタ整備後に再集計してください。

5. **他画面との役割分担**  
   - リアルタイム監視 → [検査モニタ](./inspection-monitor_ja.md)  
   - 現場入力 → [検査実績収集](./inspection-actual_ja.md)  
   - 事後補正 → [検査実績収集登録](./inspection-actual-registration_ja.md)
