成型生産計画段替予定表 項目利用ガイド

用途：教育・引継ぎ・運用ルール統一向け。A4縦でそのまま印刷できます。

1. 適用範囲

本資料は FormingInstruction.vue の「成型生産計画段替予定表」印刷内容に対応します。

現行実装では、ERP/MES ともに MES データ口径（/api/mes/forming-plan-data）を利用します。

2. ヘッダ項目（印刷上部）

項目名 表示例 データ元 用途・ルール
成型生産計画段替予定表 固定タイトル フロント固定文言 帳票名。現場掲示・保管時の識別に使用します。
生産日 2026/04/16 検索条件の日付（計画日） 本帳票の対象生産日。通常は検索開始日を表示します。
集計時間 前日15:00~当日15:00 フロント固定文言 本帳票の集計時間帯ルールを示します。
出力時刻 2026/04/16 11:42 システム現在時刻（JST） 紙帳票の出力タイミング確認用。旧版の誤使用防止に有効です。
生産計画合計数 12,800 当日計画数の合計 当日全ラインの負荷総量を管理者が即時把握するための指標です。

3. 明細項目（主テーブル）

項目名 データキー 型 表示ルール 業務上の意味・運用ポイント
生産残数(参考) totalPlanQuantity 数値 3桁区切り、空値は空白 集計時点の残数（参考）。優先順位判断や応援投入判断に使用します。
ライン line 文字列 そのまま表示 設備/ライン識別子。現場作業割当の主キーです。
予定稼働(H) plannedWorkingHours 数値（時間） 小数1桁、値域で色分け 想定稼働可能時間。色分け基準：
<8 赤、8~16 黄、16~22.5 緑、≥22.5 濃緑。0/無効値は空欄。
操業度(進捗) operationVariance 数値（差異時間） 整数表示、負値は赤字 計画と実績の差異工数。負値は遅れ、正値は先行を示します。
生産品種 productName 文字列 そのまま表示 現在または直近で生産する品種名です。
能率(本/h) efficiency 数値 そのまま表示 1時間あたり生産本数。完了見込みと排程妥当性の判断に使います。
当日計画数 planQuantity 数値 3桁区切り、強調表示 当日達成管理の中心項目です。
残生産時間 setupAfterHours 数値/文字列 そのまま表示 現状から当該品種完了までの見込み時間。引継ぎ時に重要です。
（矢印列） nextProductName 記号 次品種がある場合のみ → 次工程（次品種）への遷移があることを視覚的に示します。
次生産品種 nextProductName 文字列 空なら空欄 段替え後の次品種。段取り準備の起点情報です。
次品種計画数 nextQuantity 数値 3桁区切り、無効値は空欄 次品種の予定数。資材・金型・要員の先行準備に利用します。
備考 remarks 文字列 そのまま表示 異常、優先度、品質注意、段替え制約などを記載します。

4. 主な計算・判定ルール

  • 数量表示：3桁区切り（例：12800 -> 12,800）。
  • 予定稼働(H)：有効数値かつ 0 以外のみ表示。小数1桁。
  • 操業度(進捗)：整数表示。負値は赤字で遅れリスクを明示。
  • 矢印表示：nextProductName が非空の場合のみ表示。
  • MES口径：本画面は MES 計画データ API 基準で ERP/MES を統一運用。

5. 運用上の推奨

運用シーン 推奨対応
交替前の印刷 対象日が当日生産日であることを確認し、最新化後に印刷してください。
操業度が大幅マイナス 能率・当日計画数・実績同期の順で確認し、必要時は備考に原因と対策を記載します。
次生産品種が未設定 段替え待ちを防ぐため、計画担当が次品種を補完してください。
残生産時間が不自然 能率、可動時間設定、数量更新の反映状況を再確認してください。
紙帳票の保管 「生産日 + 班 + ライン」で保管すると、差異分析時の追跡性が向上します。

6. 印刷推奨設定

  • 用紙：A4 縦。
  • 余白：標準または狭い（本書は印刷用CSS調整済み）。
  • 拡大縮小：100%。
  • 背景グラフィックを有効化すると、表頭色や強調が保持されます。
版管理目安：v1.0（項目追加・口径変更時は本資料を同時更新してください）。
