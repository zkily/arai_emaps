## 2. 受注（月受注管理）

月次の内示・確定受注を登録・更新し、日別受注データの元となる **月受注** を整備します。後続の生産データ管理（在庫・計画列）へ反映される受注情報は、ここで整ったデータが前提です。

### 2.1 画面へのアクセス

1. Smart-EMAPs にログインします。
2. 左サイドメニューから次の順に開きます。  
   **ERP管理メニュー** → **受注管理** → **月受注管理**
3. 画面タイトル **「月受注管理」** が表示されます。

![メニューから月受注管理を開く](./images/WeldingPlanningManual/02_menu_order_monthly.png)

### 2.2 画面構成

![月受注管理 画面全体](./images/WeldingPlanningManual/07_order_monthly_overview.png)

| エリア | 説明 |
|--------|------|
| ツールバー | 日受注生成、内示更新、製品更新、日受注管理、一括登録 |
| フィルタバー | 年月、納入先、製品キーワード検索 |
| サマリーカード | 内示本数、確定本数、内示差異、社内溶接・外注溶接 等 |
| 一覧テーブル | 納入先・製品・内示本数・確定本数・操作列 |

### 2.3 初回セットアップ：月受注の登録

#### 2.3.1 一括登録（推奨）

新規月の受注をまとめて登録する場合：

1. ツールバー **「一括登録」** をクリックします。
2. ダイアログで **年・月・納入先** を選択し **「読込」** をクリックします。
3. 納入先に紐づく製品一覧が表示されます。状態が **「新規」** の行について **内示本数** を入力します。
4. 入力後 **「登録」** を実行します。

![一括登録ダイアログ](./images/WeldingPlanningManual/09_order_monthly_batch_register.png)

#### 2.3.2 個別編集

一覧から 1 件ずつ修正する場合：

1. 対象行の **「編集」** をクリックします。
2. **基本情報**（納入先・製品・タイプ）と **数量情報**（内示・日合計・差異）を確認・修正します。
3. **「保存」** をクリックします。

行末 **「削除」** で月受注レコードを削除できます（誤登録の修正時）。

### 2.4 日受注リストの生成

月受注から日別内示・確定データを展開します。

1. フィルタで **対象年月** と **納入先** を指定します（未指定の場合は確認ダイアログで表示される条件を確認）。
2. ツールバー **「日受注生成」** をクリックします。
3. 確認ダイアログで **対象年月・製品種別（量産品）・納入先** を確認します。
4. **「生成する」** をクリックし、進捗バーの完了を待ちます。

![日受注生成 確認](./images/WeldingPlanningManual/08_order_monthly_daily_generate.png)

> **注意**：日受注生成は、同一条件で再実行すると日別データが再構築されます。運用ルールに従い、必要なタイミングでのみ実行してください。

### 2.5 内示・製品情報の更新

| ボタン | 用途 | 操作概要 |
|--------|------|----------|
| **内示更新** | 内示本数の一括再計算 | 確認ダイアログ → **実行** |
| **製品更新** | 製品マスタ変更の反映 | 開始日を指定 → **製品情報を最新データに更新** にチェック → **更新実行** |
| **日受注管理** | 日別受注の詳細編集 | 別画面（日受注管理）へ遷移 |

### 2.6 サマリーと溶接関連の確認

画面上部のサマリーカードで、当該月の **社内溶接**・**外注溶接** の件数（本数）を確認できます。計画作成前に、対象製品の受注が登録されているか、このサマリーと一覧で照合してください。

### 2.7 受注工程チェックリスト

| # | 確認内容 | OK |
|---|----------|-----|
| 1 | 対象年月・納入先の月受注が登録済みか | ☐ |
| 2 | 内示本数・確定本数に大きな未入力がないか | ☐ |
| 3 | 日受注生成を実行済みか（日別データが必要な場合） | ☐ |
| 4 | 社内溶接対象製品が一覧に含まれているか | ☐ |

## 3. 在庫（生産データ管理）

受注・実績・在庫を横断的に管理する画面です。計画作成の前提として、**溶接在庫**・**溶接前在庫**・**溶接計画** 列などが正しく更新されている必要があります。

### 3.1 画面へのアクセス

1. **ERP管理メニュー** → **生産管理** → **生産計画** → **生産データ管理**
2. 画面タイトル **「生産データ管理」**、サブタイトル **「受注・実績・在庫を一元管理」** が表示されます。

![メニューから生産データ管理を開く](./images/WeldingPlanningManual/03_menu_data_management.png)

### 3.2 画面構成

![生産データ管理 画面全体](./images/WeldingPlanningManual/10_data_management_overview.png)

| エリア | 説明 |
|--------|------|
| ヘッダー | 件数表示、各種更新機能、再取得、印刷、列設定 等 |
| フィルタ | 年月・製品・納入先 等（画面上部） |
| メインテーブル | 日付行 × 工程別列（受注・実績・在庫・計画 等） |
| 下部タブ | 受注／実績／**在庫**／推移／計画 等 — 表示列グループの切替 |

### 3.3 初回：データ生成と受注反映

初めて当該期間の生産データを扱う場合：

1. 右上 **「各種更新機能」** ▼ をクリックします。
2. 次の順序で実行することを推奨します。

| 順序 | メニュー項目 | 目的 |
|------|--------------|------|
| 1 | **データ生成** | 対象期間の行を生成 |
| 2 | **受注データ更新** | 月受注・日受注から受注列を反映 |
| 3 | **初期在庫一括登録** | 期首在庫がある場合のみ |
| 4 | **在庫・推移更新** | 在庫・推移・安全在庫を再計算 |

![各種更新機能メニュー](./images/WeldingPlanningManual/12_data_management_update_menu.png)

#### 3.3.1 全部一括更新

上記を個別に実行する代わりに、**「全部一括更新」** 1 回で以下を順に実行できます。

- 受注データ更新
- 繰越データ更新
- 実績・不良・廃棄・保留 更新
- 生産日・設備 更新
- 計画データ更新
- 在庫・推移・安全在庫更新

確認ダイアログの内容を読み、**「一括更新開始」** をクリックします。進捗ダイアログで完了を待ちます。

![全部一括更新（参考）](./images/WeldingInstructionManual/14_all_batch_update.png)

### 3.4 在庫 TAB の操作

計画作成前に **溶接在庫** の状態を確認します。

1. 画面下部タブ列から **「在庫」**（📦）をクリックします。
2. テーブルに **溶接在庫** 等の列が表示されます。
3. 日付行と製品行の交差セルで、在庫数量・推移を確認します。

![在庫 TAB](./images/WeldingPlanningManual/11_data_management_inventory_tab.png)

#### 3.4.1 在庫の手動修正（必要な場合）

実物在庫とシステム在庫に差がある場合：

1. 修正したい **日付行** と **工程列**（例：溶接在庫）のセルを **ダブルクリック** します。
2. **「在庫取引ログ入力」** ダイアログが開きます。
3. 種別・数量を入力し **登録** します。
4. 修正後、**「在庫・推移更新」** を実行して関連列を再計算します。

![在庫セル ダブルクリック](./images/WeldingInstructionManual/15_inventory_tab_dblclick.png)

### 3.5 溶接関連列の見方

| 列名 | 意味 |
|------|------|
| 溶接前在庫 | 溶接直前工程の在庫（投入可能数の参考） |
| 溶接在庫 | 溶接工程完成品の在庫 |
| 溶接実績 | 溶接完成実績数 |
| 溶接計画 | 計画数量（APS／ERP 連携） |
| 溶接実計 | 計画と実績の整合列 |
| 溶接推移 | 在庫推移指標 |

### 3.6 在庫工程チェックリスト

| # | 確認内容 | OK |
|---|----------|-----|
| 1 | 受注データ更新後、受注列に数値が入っているか | ☐ |
| 2 | 溶接在庫・溶接前在庫が妥当か（負数・異常な跳びがないか） | ☐ |
| 3 | 在庫 TAB で対象製品の直近数日を確認したか | ☐ |
| 4 | 修正後は在庫・推移更新を実行したか | ☐ |

## 4. 計画（溶接計画作成）

APS 上で溶接ラインの生産計画を作成・編集し、ガント表示で日程を確認します。

### 4.1 画面へのアクセス

1. **APS管理メニュー** → **生産計画作成** → **溶接計画作成**
2. 画面タイトル **「溶接計画作成」** が表示されます。

![メニューから溶接計画作成を開く](./images/WeldingPlanningManual/04_menu_welding_planning.png)

### 4.2 画面構成

![溶接計画作成 画面全体](./images/WeldingPlanningManual/13_welding_planning_overview.png)

| エリア | 説明 |
|--------|------|
| セットアップバー | 基準開始月、工程、ライン、計画を取得 |
| 計画追加 | 製品・ロット数／本数・追加先（新規行／既存合算） |
| 計画一覧 | 順位・製品名・合計(本)・開始日・終了日 等 |
| ガント | 日別計画の視覚表示 |
| アンカー日設定 | 再計算開始日の設備別保存 |

### 4.3 計画取得の手順

1. **基準開始月** を設定します（先頭計画の着手月）。
2. **工程** を選択します（溶接工程 CD）。
3. **ライン**（溶接設備）を選択します。  
   ※ 工程に紐づく設備が設備マスタ・設備能率に登録されている必要があります。
4. **「計画を取得」** をクリックし、既存計画を読み込みます。

必要に応じて **「アンカー日設定」** で設備ごとの再計算開始日を保存できます（保存済みは最優先で使用されます）。

### 4.4 計画の追加

![計画追加フォーム](./images/WeldingPlanningManual/14_welding_planning_add_plan.png)

1. **製品名** を選択します（設備能率に登録された製品のみ表示）。
2. 製品選択後、**能率・段取・ロットサイズ・最大日産** がチップ表示されます。
3. **入力方式** を選びます。
   - **ロット数**：ロット単位で追加
   - **本数**：直接本数を指定
4. **追加先** を選びます。
   - **新規行**：計画一覧に新しい行を追加
   - **既存に合算**：同一製品の既存行に数量を加算
5. 数量を入力し **「追加」** をクリックします。

### 4.5 計画一覧の編集

| 操作 | 方法 |
|------|------|
| 順位変更 | 行をドラッグ＆ドロップ（「未完のみ」表示時） |
| 合計(本) 修正 | セルをダブルクリック → 数値入力 → 保存 |
| 開始日指定 | 列「開始日指定」をダブルクリックで日付設定 |
| 削除 | 行末の削除操作 → 確認 |

### 4.6 ライン順で再計算

計画追加・順位変更・数量修正後は、必ず再計算します。

1. 計画一覧右上 **「ライン順で再計算」**（オレンジ）をクリックします。
2. 確認ダイアログの内容を確認します。  
   - 過去日の日別計画は固定  
   - 当日は実績がある場合のみ固定  
   - 開始日指定がある製品は指定日より前に開始しない
3. 実行を承認し、完了メッセージを待ちます。
4. ガントと計画一覧で開始日・終了日・日別数量を確認します。

![ライン順で再計算](./images/WeldingPlanningManual/15_welding_planning_replan.png)

### 4.7 計画工程チェックリスト

| # | 確認内容 | OK |
|---|----------|-----|
| 1 | 正しい工程・ラインを選択しているか | ☐ |
| 2 | 計画数量（合計本数）が受注・在庫バランスと整合しているか | ☐ |
| 3 | 再計算後、ガントに日別計画が表示されているか | ☐ |
| 4 | 生産残（未完了計画）が意図どおりか | ☐ |

## 5. 指示（溶接指示）

APS で確定した計画に基づき、現場向け **溶接生産指示書** を発行します。

### 5.1 画面へのアクセス

1. **MES管理メニュー** → **生産指示** → **溶接指示**
2. 画面タイトル **「溶接指示書発行管理」** が表示されます。

![メニューから溶接指示を開く](./images/WeldingPlanningManual/05_menu_welding_instruction.png)

### 5.2 画面構成

![溶接指示書発行管理 画面全体](./images/WeldingInstructionManual/03_welding_instruction_overview.png)

| エリア | 説明 |
|--------|------|
| ヘッダー | 計画生産数・稼働設備数、**データ更新** |
| 検索バー | 生産日、設備、キーワード、印刷ボタン群 |
| 計画テーブル | 生産日・設備・製品・計画生産数・能率 等 |
| チャート | 日別計画・実績 |

### 5.3 データ更新

計画作成後、指示画面に最新計画を反映します。

1. 画面右上 **「データ更新」** をクリックします。
2. 更新完了後、計画テーブルとチャートを確認します。

### 5.4 指示書発行

> **重要**：**原則として翌日の生産日** を選んでから発行してください。

1. 検索バーで **生産日** を **翌日** に設定します。

![生産日を翌日に設定](./images/WeldingInstructionManual/04_welding_date_nextday.png)

2. 必要に応じて **設備**・**キーワード** で絞り込みます。
3. **「指示書発行」**（緑色）をクリックします。
4. 印刷プレビュー／QR 読取画面で内容を確認し、印刷します。  
   用紙は **A5 横向き** を指定してください。

![指示書発行ボタン](./images/WeldingInstructionManual/05_instruction_issue_button.png)

### 5.5 段取予定発行（任意）

現場掲示用に **溶接生産計画段取予定表** を印刷する場合：

1. 検索条件（生産日・設備）を設定します。
2. **「段取予定発行」**（青色）をクリックします。
3. 印刷物を現場に掲示します。

詳細は [溶接工程 生産指示・実績収集マニュアル](./welding-instruction) 第 9 章を参照してください。

### 5.6 指示工程チェックリスト

| # | 確認内容 | OK |

## 6. 実績（ハンディターミナル・MES）

現場で溶接実績を登録します。**ハンディターミナル（HT）** または **MES 溶接実績収集** 画面を使用します。

### 6.1 MES 溶接実績収集

#### 6.1.1 画面へのアクセス

1. **MES管理メニュー** → **実績収集** → **溶接実績収集**
2. 画面タイトル **「溶接実績収集」** が表示されます。

![メニューから溶接実績収集を開く](./images/WeldingPlanningManual/06_menu_mes_welding_actual.png)

#### 6.1.2 操作手順

![MES 溶接実績収集画面](./images/WeldingPlanningManual/18_mes_welding_actual.png)

1. **生産日** を設定します（前日／今日／翌日ボタン可）。
2. **溶接設備** を選択します。
3. **溶接製品** を選択します（バーコード **読取** も可）。
4. **溶接作業者** を指定します。
5. **生産開始** → 作業中は **一時停止**／**再開** → **生産終了** → **実績を確定** の順で操作します。
6. 不良がある場合は項目別に +/- で入力します。

| ボタン | 用途 |
|--------|------|
| 生産開始 | 稼働開始を記録 |
| 一時停止／再開 | 中断・再開 |
| 生産終了 | 稼働終了 |
| 実績を確定 | welding_management へ確定保存 |

> **注意**：別製品が生産中の設備では、先に **生産終了** してから製品を切り替えてください。

### 6.2 ハンディターミナル（HT）

タブレット／専用端末で HT アプリから実績を入力する運用も可能です。

| 操作 | 内容 |
|------|------|
| 起動 | 端末の電源投入・初期画面 |
| ログイン | 担当者でサインイン |
| 実績入力 | 製品・設備・数量の登録 |
| 送信 | 入力内容の確定と送信 |

![HT 操作（参考）](./images/WeldingInstructionManual/09_handy_terminal.png)

HT 入力後は、ERP **生産実績管理** で HT・指示書と照合します（[溶接指示・実績収集マニュアル](./welding-instruction_ja.md) 第 5 章）。

### 6.3 実績工程チェックリスト

| # | 確認内容 | OK |
|---|----------|-----|
| 1 | 生産日・設備・製品が指示書と一致しているか | ☐ |
| 2 | 生産開始〜終了が漏れなく操作されたか | ☐ |
| 3 | 不良数量を入力したか（該当時） | ☐ |
| 4 | オフライン時は同期完了を確認したか | ☐ |

## 7. 日次サイクルの繰り返し

初回の **受注 → 在庫 → 計画 → 指示 → 実績** が完了したら、以降は次の **4 ステップ** を毎営業日繰り返します。

```
在庫（確認・更新）→ 計画見直し（再計算）→ 指示（翌日分）→ 実績（収集・照合）
        ↑__________________________________________________________|
                              繰り返し
```

### 7.1 在庫（確認・更新）

1. **生産データ管理** を開き、前日実績反映後の **溶接在庫**・**溶接実績** を確認します。
2. 生産実績管理で照合・修正が済んでいれば、**「全部一括更新」** または **「在庫・推移更新」** を実行します。
3. 微差がある場合は **在庫 TAB** から取引ログで修正します。

![日次サイクル：在庫と計画](./images/WeldingPlanningManual/20_cycle_inventory_replan.png)

### 7.2 計画見直し（再計算）

1. **溶接計画作成** または **溶接計画一覧**（`/aps/welding-planning-list`）を開きます。
2. 生産残・合計(本) を確認し、必要なら数量を修正します。
3. **「ライン順で再計算」**（計画作成）または **「溶接ライン順で再計算」**（計画一覧）を実行します。
4. ガントで残計画の日程を確認します。

詳細手順は [溶接指示・実績収集マニュアル 第 8 章](./welding-instruction_ja.md#8-aps-再計算と生産残の修正) を参照してください。

### 7.3 指示（翌日分）

1. **溶接指示** で **データ更新** を実行します。
2. 生産日を **翌日** に設定し、**指示書発行** します。
3. 現場へ指示書を配布します。

### 7.4 実績（収集・照合）

1. 現場で HT／MES により実績を登録します。
2. 締め時に **生産実績管理** で照合します。
3. **全部一括更新** で DB へ反映します。
4. 翌朝、**7.1 在庫** からサイクルを再開します。

### 7.5 日次サイクル早見表

| 時期 | 在庫 | 計画 | 指示 | 実績 |
|------|------|------|------|------|
| 前日締め後 | 一括更新 | — | — | 照合完了 |
| 当日朝 | 在庫確認 | 再計算 | — | — |
| 当日昼〜夕 | — | 残数修正 | 翌日指示発行 | — |
| 当日生産中 | — | — | 現場配布 | MES／HT 入力 |
| 当日締め | 更新準備 | — | — | 照合 |

## 8. よくあるメッセージと対処

| メッセージ・現象 | 原因の例 | 対処 |
|------------------|----------|------|
| 計画データがありません | ライン未選択／未登録 | 工程・ラインを選び計画を追加 |
| この工程に該当する設備がありません | 設備マスタの種別不一致 | 設備マスタ・APS 対象フラグを確認 |
| 製品は工程に属していません | BOM ルートに工程なし | 製品工程 BOM を確認 |
| 順次再計算に失敗 | 稼働カレンダー・能率未設定 | 設備能率・稼働時間を確認 |
| 指示書に計画が出ない | 指示画面が未更新 | **データ更新** を実行 |
| 実績が在庫に反映されない | 一括更新未実行 | 照合後 **全部一括更新** |
| 内示差異が大きい | 日受注未生成・未更新 | 月受注・日受注生成を確認 |
