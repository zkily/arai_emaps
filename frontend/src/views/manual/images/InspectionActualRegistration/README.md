# 検査実績収集登録 — スクリーンショット用フォルダ

操作説明書 `frontend/src/views/manual/docs/inspection-actual-registration_ja.md` から参照する画像を、このフォルダに配置してください。

## 推奨ファイル名一覧

| ファイル名 | 内容 |
|-----------|------|
| `01_overview.png` | 画面全体（ヘッダー・実績入力パネル・登録一覧） |
| `02_menu_navigation.png` | サイドメニュー「MES実績収集登録」→「検査実績登録」 |
| `03_production_day.png` | ①生産日（日付ピッカー・前日/今日/翌日ボタン） |
| `04_quantity.png` | ④生産数（箱数×入数=本数、派生セルの破線枠） |
| `05_time.png` | ⑤生産時間（開始/終了/休憩/停止・シフトプレビュー） |
| `06_defects.png` | ⑥不良（帰属工程グループ・+/- 入力） |
| `07_list_table.png` | 登録一覧テーブル（取得元・能率・操作列） |
| `08_workflow.png` | 新規登録標準フロー図（§7） |
| `08_workflow.mmd` | 上記フロー図の Mermaid ソース（再生成用） |
| `09_edit_mode.png` | 編集モード（編集中 #ID・製品名読取専用） |
| `10_delete_confirm.png` | 削除確認ダイアログ（警告付き） |

## 撮影のヒント

- 解像度：横幅 1280px 以上推奨
- テストデータ：仕上課の検査員・入数設定済み製品で撮影
- **ロックヒント**：検査員未選択・製品未選択の状態も別途撮影すると説明に有用
- **編集モード**：一覧から行を読込み、「編集中 #」表示を確認
