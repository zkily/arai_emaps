# 検査実績収集 — スクリーンショット用フォルダ

操作説明書 `frontend/public/docs/inspection_actual_manual_ja.md` から参照する画像を、このフォルダに配置してください。

## 推奨ファイル名一覧

| ファイル名 | 内容 |
|-----------|------|
| `01_overview.png` | 画面全体（タイトル・ツールバー・生産カード・確定実績） |
| `02_menu_navigation.png` | サイドメニューで「MES実績収集」→「検査実績収集」を開く様子 |
| `03_toolbar.png` | 生産日・検査員・検査製品・読取ボタン |
| `04_locale_switch.png` | 右上 JP / EN / ZH / VI 言語切替 |
| `05_empty_select_product.png` | 製品未選択時の空状態 |
| `06_in_progress_strip.png` | 「当日 検査生産中」チップ列 |
| `07_production_card_idle.png` | 生産カード（未開始・タイマー idle） |
| `08_production_card_running.png` | 生産開始後（計測中・不良パネル） |
| `09_production_card_paused.png` | 一時停止中 |
| `10_defect_stepper.png` | 不良項目別 +/- 入力のクローズアップ |
| `11_end_dialog.png` | 生産終了・実績確定ダイアログ |
| `12_history_table.png` | 本日の確定実績テーブル |
| `13_print_preview.png` | 印刷プレビューまたは印刷ダイアログ |
| `14_confirmed_edit_dialog.png` | 確定実績の修正ダイアログ |
| `15_barcode_scan_dialog.png` | バーコード / QR 読取ダイアログ |
| `16_session_recovery_alert.png` | 未終了セッション復旧アラート |
| `17_offline_banner.png` | オフライン／未同期バナー |
| `18_other_terminal_lock.png` | 「他端末生産中」表示 |

## 撮影のヒント

- 解像度：横幅 1280px 以上推奨（タブレット実機は 1024px でも可）
- 個人情報：検査員の実名が写る場合はマスクまたはテストユーザーで撮影
- ブラウザ：Chrome / Edge の最新版、本番と同じ HTTPS 環境
- ダークモード：社内標準のテーマに合わせて 1 セットに統一
