## 実機撮影のヒント（差し替え時）

- 端末：現場で使用する Android タブレット（横向き／縦向きは実運用に合わせる）
- 解像度：幅 1080px 以上推奨
- 個人情報：検査員名はテストユーザーまたはマスク
- 入数設定済み製品で終了ダイアログを撮影（箱数 × 入数 = 本数）
- 不良は帰属工程ごとの色分けグループが分かるように撮影
