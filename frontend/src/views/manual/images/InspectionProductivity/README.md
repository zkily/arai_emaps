# 検査工程 — 生産性分析 — スクリーンショット用フォルダ

## 推奨ファイル名一覧

| ファイル名 | 内容 |
|-----------|------|
| `01_overview.png` | 画面全体（KPI・チャート・ランキング） |
| `02_menu_navigation.png` | MES → 実績分析項目 → 生産性分析 → 検査生産性 |
| `03_filters.png` | 期間・検査員・製品・未確定チェック |
| `04_kpi.png` | KPI 5 カード |
| `05_daily_chart.png` | 日別推移チャート |
| `06_inspector_panel.png` | 検査員別チャート＋表 |
| `07_product_panel.png` | 製品別チャート＋表 |
| `08_weld_rank.png` | 溶接なし/あり 能率ランキング |
| `09_product_rank.png` | 製品別検査員能率（表彰台含む） |
| `10_metrics_time.png` | 検査員別指標・時間 |
| `11_metrics_defect.png` | 検査員別指標・不良内訳 |
| `12_sessions.png` | セッション明細＋CSV |
| `13_report_menu.png` | レポートドロップダウン |

## 撮影のヒント

- 直近 30 日など実データがある期間で撮影
- ランキング・表彰台は複数検査員・複数製品がある状態が望ましい
- レポートメニューはデータ読込後に開いて撮影
